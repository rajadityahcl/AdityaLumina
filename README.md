# 🌟 Aditya Lumina (ElevatePath Previously named)
### Clarity. Direction. Achievement.

Aditya Lumina — also known as **ElevatePath** — is a modern, hybrid **Career Acceleration Platform** designed for online students, international learners, and career switchers.  
It blends **Human Coaching + AI-powered tools** to deliver personalized guidance, powerful insights, and real-world career readiness.

---

## 🚀 Vision
To empower learners with clarity, direction, and actionable strategies using a combination of human mentoring and intelligent automation — helping them land internships, full-time roles, and meaningful opportunities faster.

---

## 🎯 Core Features

### 🔹 Personalized Career Roadmap
- Skill gap analysis  
- Weekly action plans  
- Milestone tracking  
- Project recommendations  

### 🔹 Resume & LinkedIn Optimization
- ATS-friendly resume rewrites  
- Keyword-aligned job targeting  
- Portfolio & GitHub review  
- Data-driven scoring (AI assisted)  

### 🔹 Mock Interviews + Feedback
- Behavioral and technical sessions  
- SQL, Python, Data Analyst, and case-based practice  
- Recorded notes & feedback  
- Confidence + communication coaching  

### 🔹 AI Tools Suite
- Resume scoring tool  
- Job description match engine  
- Interview question generator  
- Study planner automation  
- Career roadmap generator  

### 🔹 Payment + Booking Integrations
- Integrated **UPI payment link**  
- **Calendly** meeting scheduling  
- Tier-based pricing model (Free → Elite)
